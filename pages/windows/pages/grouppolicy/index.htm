Group Policy

There are two types of Group Policy in Windows Server. Local Group Policy and Domain Group Policy.

Local Group Policy is used for local only policies. So policies are only able to be applied to a specific machine and cannot be easily managed.

Domain Group Policy uses the same policies (and more!) as Local Group Policy, except it is to be ran on an Active Directory server, with Users and Computers.

Features of Domain Group Policy over Local Group Policy.

Domain Group Policy can be:

Managed from one central server.

Can do drive mapping.

Can do registry key management.

Can do software managment.

Can list and choose from forests inside of a domain.


A forest is a way to group and divide on a domain. So that you can have multiple instances of Active Directory be separate on the same domain.
