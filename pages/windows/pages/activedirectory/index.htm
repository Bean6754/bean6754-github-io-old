Active Directory

Active Directory is a service inside of Windows Server. It is used for controlling Users and Groups from within a domain.

An Active Directory server is often refered to as a Domain Controller. This is because a Domain Controller manages and controls the elements of a domain infrastructure.

Read more on Domain Controllers here.

Can list and choose from forests inside of a domain.


A forest is a way to group and divide on a domain. So that you can have multiple instances of Active Directory be separate on the same domain.
